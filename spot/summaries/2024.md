# 📅 Thay đổi năm 2024

> Nguồn: https://developers.binance.com/docs/binance-spot-api-docs/CHANGELOG

## 2024-12-17

**Microsecond support** được thêm vào tất cả time/timestamp fields (opt-in, mặc định vẫn là milliseconds).

### WebSocket Streams
- Parameter `timeUnit` mới trong connection URL (giá trị: `millisecond` / `microsecond`).

### REST API
- Header `X-MBX-TIME-UNIT` mới.
- Timestamp params (`startTime`, `endTime`, `timestamp`) hỗ trợ cả milliseconds lẫn microseconds.

### WebSocket API
- Parameter `timeUnit` mới trong connection URL.

### User Data Streams
- Parameter `timeUnit` mới trong connection URL.

## 2024-12-09 (Rollout từ 2024-12-12, ~1 tuần)

### General
- Timestamp params từ chối giá trị quá xa trong quá khứ (trước 2017-01-01) hoặc tương lai (>10 giây).
- Field `origQuoteOrderQty` được thêm vào responses của nhiều endpoints:
  - REST: `POST /api/v3/order`, `DELETE /api/v3/order`, `POST /api/v3/order/cancelReplace`, v.v.
  - WebSocket: `order.place`, `order.cancel`, `order.cancelReplace`, v.v.

### SBE
- **Schema 2:1** ra mắt; schema 2:0 deprecated (retire sau 6 tháng).
- Schema 2:1 thêm `origQuoteOrderQty` trong order responses, và các messages User Data Stream.

### WebSocket API
- **Subscribe User Data Stream** ngay qua WebSocket connection (cần Ed25519 key + SBE schema 2:1).
- Methods mới: `userDataStream.subscribe`, `userDataStream.unsubscribe`.
- Session requests có thêm field `userDataStream`.

### User Data Stream
- Event mới: `eventStreamTerminated` (khi logout hoặc unsubscribe).
- Event mới: `externalLockUpdate` (khi spot wallet bị lock/unlock bởi hệ thống bên ngoài).

### FIX API
- Schema cập nhật thêm Administrative message `News <B>` (thông báo trước khi connection bị đóng).

> **Tại 2024-12-16 đến 2024-12-20:**
- Fix bug với OCO `BUY` side thiếu `stopPrice`.
- **TAKE_PROFIT** và **TAKE_PROFIT_LIMIT** hỗ trợ OCO.

## 2024-04-10 → 2024-04-25

### General (breaking change)
- Permission information chuyển từ field `permissions` → `permissionSets`.
- `permissions` sẽ bị xóa trong tương lai.
- `permissionSets` hỗ trợ logic phức tạp hơn (AND giữa các sets).

### REST API
- `otoAllowed` xuất hiện trong `GET /api/v3/exchangeInfo`.

### WebSocket API
- `otoAllowed` xuất hiện trong `exchangeInfo`.

### SBE
- **Schema 2:0** ra mắt; schema 1:0 deprecated (retire sau 6 tháng).

## 2024-04-02 (Rollout ~1 tuần)
